Os conjuntos em python não permitem a duplicidade de elementos, logo ele pode ser usado para remover duplicatas de uma lista sem que seja necessário percorrer essa lista com um for. A identificação de elementos únicos se dá através dessa característica do conjunto não aceitar elementos duplicados, logo todos os elementos do conjunto são únicos. Os conjuntos em python tem funções úteis de conjuntos como a verificação de pertencimento que pode ser usada para verificar se um elemento está contido dentro de um conjunto de elementos. Outra possibilidade é a manipulação de dados distintos em dois conjuntos que podem fazer uso da união, intersecção e diferença desses conjuntos.
