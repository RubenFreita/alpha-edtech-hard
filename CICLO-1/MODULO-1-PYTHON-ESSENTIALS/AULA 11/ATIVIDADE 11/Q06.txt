As tuplas são ideias para representação de uma sequência de DNA, por ser imutável e assim garantir a integridade da sequência de DNA, porém também é possível se usar listas para representar as sequências de DNA, entretanto é preciso ter um cuidado a mais para garantir a integridade, visto que as listas são mutáveis. Outra opção é usar strings já que as mesmas também são imutáveis, mas é possível adicionar elementos na string o que facilitaria para a não integridade então também seria necessário um cuidado a mais, portanto eu acredito que as tuplas é a estrutura de dados mais ideal para se representar uma sequência de DNA.
